## Notes

- Auto-grouping is **silent** (doesn't block upload response)
- Auto-grouping is **best effort** (fails don't rollback upload)
- Subscription is **fire-and-forget** (publish doesn't await completion)
- Batch processing is **asynchronous** (doesn't block Lambda timeout)
- First upload processes **all historical photos**
- Subsequent uploads process **just the new photo**
